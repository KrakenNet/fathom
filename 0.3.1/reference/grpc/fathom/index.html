    FathomService¶

    Method Name Request Type Response Type Description
    Evaluate EvaluateRequest EvaluateResponse Evaluate asserted facts against loaded rules and return a decision.
    AssertFact AssertFactRequest AssertFactResponse Assert one or more facts into working memory.
    Query QueryRequest QueryResponse Query working memory for facts matching a template and optional filter.
    Retract RetractRequest RetractResponse Retract facts matching a template and optional filter.
    SubscribeChanges SubscribeRequest FactChange stream Stream working-memory changes as they occur during evaluation.
