# `fathom test`

```
                                                                                                    
 Usage: fathom test [OPTIONS] RULES_PATH TEST_PATH                                                  
                                                                                                    
 Run YAML test cases against a rule pack.                                                           
                                                                                                    
┌─ Arguments ──────────────────────────────────────────────────────────────────────────────────────┐
│ *    rules_path      PATH  Path to a rule pack directory. [required]                             │
│ *    test_path       PATH  Path to a YAML test file or directory of test files. [required]       │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘
┌─ Options ────────────────────────────────────────────────────────────────────────────────────────┐
│ --help          Show this message and exit.                                                      │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘
```
